# 🎬 MovieLens Data Analysis & Dashboard (Power BI)

## 📊 Business Problem
Analyze movie ratings data to understand user preferences, genre popularity, and trends across demographics to support data-driven entertainment insights.

---

## 🎯 Objective
- Analyze movie ratings and user behavior  
- Identify popular genres and trends  
- Understand demographic patterns  
- Build an interactive dashboard for insights  

---

## 🛠 Tools Used
- Power BI (DAX, Data Modeling, Visualization)
- Excel / CSV Dataset

---

## 📁 Dataset Overview
The dataset includes:
- Movies and genres  
- User demographics (age, gender, occupation)  
- Ratings data  

---

## 🔍 Key Analysis

- Number of movies and genres  
- Ratings distribution across demographics  
- Popular genres  
- Trends over years  
- User behavior patterns  

---

## 📈 Key Insights

- Certain genres consistently receive higher ratings  
- Ratings vary across different age groups  
- Male and female users show different genre preferences  
- Classic movies (before 2000) still maintain strong ratings  

---

## 💡 Business Recommendations

- Focus on high-performing genres for content strategy  
- Personalize recommendations based on demographics  
- Promote classic content for niche audiences  
- Use rating trends to guide content acquisition
